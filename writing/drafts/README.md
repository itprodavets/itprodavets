# drafts/

`/write-post` drops drafts here as `YYYY-MM-DD-slug.md` with frontmatter.

A draft is not done until it clears the depth bar in [`../style-guide.md`](../style-guide.md)
— ≥1 real tradeoff, ≥1 concrete specific, ≥1 transferable insight. Edit it into your
voice, then move it to [`../posts/`](../posts/).
